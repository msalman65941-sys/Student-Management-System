# Student Management System (Java OOP Project)

## 📌 Project Overview
The **Student Management System** is a console-based application developed in **Java using Object-Oriented Programming (OOP) principles** as part of my second semester project.

This system is designed to manage students, teachers, departments, attendance, results, fees, and other academic activities in a structured and organized way.

It includes a secure login system and a menu-driven interface for easy navigation.

---

## 🔐 Security Feature
- Secure **Sign-in system (Login with ID & Password)**
- Role-based access control (Admin/User)
- Only authorized users can access system features

---

## 🧠 OOP Concepts Used
This project is fully built using OOP principles:

- Abstraction
- Inheritance
- Encapsulation
- Polymorphism

---

## 📚 Classes Structure

The system consists of **11 classes**:

### 🧩 Core Classes
- **Person (Abstract Class)**
- **Student (extends Person)**
- **Teacher (extends Person)**
- **Main Class (System Controller)**

### 🏫 Functional Classes
- Course (Teacher has Course relationship)
- Department
- Attendance
- Result
- Fee
- Scholarship
- Notification

---

## ⚙️ Features

### 👨‍🎓 Student Management
- Enroll students
- Update student information
- View student details

### 👨‍🏫 Teacher Management
- Add and manage teachers
- Assign courses to teachers

### 🏫 Academic Management
- Manage departments
- Assign courses
- Track attendance
- Manage results
- Handle fee records
- Scholarship management system

## 📋 Menu System

- Enroll students

- Manage departments
- Navigate system easily through options

## 🎯 Project Purpose

## 🚀 How It Works
1. Run the main class
2. Login using credentials
3. Choose role (Admin/User)
4. Navigate through menu options
5. Perform operations like enroll, attendance, fee management, etc.
